# 🏫 Advanced School Management System (ASMS)

A web-based School Management System built using **Flask (Python)** and **MySQL**.  
It helps schools manage students, teachers, exams, and results in one centralized system.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Add new students
- View all students
- Edit student details
- Delete students

### 👨‍🏫 Teacher Management
- Add teachers
- View teachers
- Edit teacher details
- Delete teachers

### 📝 Exam Management
- Create exams
- View exams
- Edit exams
- Delete exams

### 📊 Results Management
- Add student results
- View results
- Edit results
- Delete results
- Auto grading system (A, B, C, F)

### 📈 Analytics
- Results statistics (A, B, C, F count)
- Dashboard summary cards

### 🔐 Authentication System
- User registration
- Login system
- Session management
- Role-based access (Admin / Teacher)

---

## 🛠️ Tech Stack

- Python (Flask)
- MySQL Database
- HTML5 / CSS3
- Jinja2 Templates
- Werkzeug Security
